Поддерживаемые форматы офлайн-словарей
======================================

Можно импортировать словари через: Настройки -> Офлайн-словари -> Импортировать словарь.

1) CSV или TSV
---------------
Минимум две колонки:
word,translation,partOfSpeech
hello,"привет, здравствуйте",interjection
beautiful,"красивый, прекрасный",adjective

TSV также поддерживается:
hello	привет; здравствуйте	interjection

2) JSON object
--------------
{
  "hello": "привет, здравствуйте",
  "beautiful": ["красивый", "прекрасный"]
}

3) JSON array
-------------
[
  {"word":"hello", "translationsRu":["привет", "здравствуйте"], "partOfSpeech":"interjection"},
  {"word":"maison", "translationRu":"дом", "partOfSpeech":"noun"}
]

4) DSL / Lingvo-style plain text
--------------------------------
Поддерживается базовый DSL: строка без отступа = слово, строки с отступом = перевод/описание.
Сложная разметка очищается упрощённо.

Где взять словари
=================

Лучше использовать открытые источники с понятной лицензией:
- FreeDict: EN-RU и FR-RU существуют в пакетах dictd/GoldenDict-совместимых форматов.
- Debian пакеты dict-freedict-eng-rus и dict-freedict-fra-rus.
- Kaikki/Wiktionary данные можно конвертировать в JSON/CSV, но они крупнее и требуют обработки.

Важно: не импортируйте словари из APK/iOS-приложений без права на повторное использование данных.
